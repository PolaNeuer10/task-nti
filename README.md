# NTI Web Design Creative Grant - Tasks

This repository contains the practical tasks submitted for the **NTI Web Design Creative Grant**. Each task demonstrates different skills in Front-end development (HTML, CSS, and Logic).

---

## 📂 Project Tasks

### 1️⃣ Registration Form (Task 1)
* **Description:** A professional user registration form.
* **Tech used:** HTML5, CSS3.
* **Key Features:** Form validation, responsive design, and clean UI for user data entry.

### 2️⃣ Numbers Visual Layout (Task 2)
* **Description:** A creative UI task that displays numbers from "Ten" down to "One" in a specific visual hierarchy.
* **Key Features:** * Centered typography with custom horizontal dividers.
    * Focus on CSS layout, spacing, and alignment to match a specific design template.

### 3️⃣ Calculator Project (Task 3)
* **Description:** A functional web-based calculator.
* **Tech used:** HTML, CSS, and JavaScript.
* **Key Features:** Basic mathematical operations with a clean and interactive interface.
